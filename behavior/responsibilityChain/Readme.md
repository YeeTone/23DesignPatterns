## 1. 为什么要学习责任链模式？
有时候，对于一套连贯的处理流程来说，       
我们并不希望期间涉及的所有对象都像门面模式那样全部堆在一块，使得步骤中的每一个流程都能对任一流程都可以访问；         
而是希望对于其中任何一个流程来说，仅仅可以对其的下一个流程进行访问。        
       
这时候，责任链模式可以将流程中的多个步骤进行解耦分离，按照链式结构依次访问 **（类似数据结构中的单向链表）**，从而简化程序结构。
