## 1. 为什么要学习外观模式？
有时候，一个系统的完整调度需要涉及好几个子系统的共同协同工作，共同协作调度过程极为复杂。      
这时候，外观模式就能够将复杂的协同工作聚合起来，将系统的复杂性对上层屏蔽处理，而仅仅需要调用一个自定义接口即可完成所有工作。
